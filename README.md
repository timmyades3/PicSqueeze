# PicSqueeze - Your Image Manipulation Toolkit

## Features

- **Image Compression:** Reduce the file size of your images while preserving quality.
- **Image Format Conversion:** Convert images between various formats, such as JPEG, PNG, GIF, and more.
- **Image Resizer:** Resize images to specific aspect ratios, suitable for different platforms and devices.

## How to Use

### Step 1: Launch the App

**Launch:** To get started, simply click the link below to launch PicSqueeze:

[![PicSqueeze App](https://i.postimg.cc/RCpRsS5r/button-launch.png)](http://timmyades3.pythonanywhere.com/)

### Step 2: Exploring the Main Screen

Upon launching the app, you will be redirected to the main screen, which provides access to the image compression feature and links to the Image Format Conversion and Image Resizer features. If you're not logged in, you will be directed to the login page.

- **Image Compression:** Click this option to reduce the file size of your images while preserving quality.

- **Image Format Conversion:** Choose this option to convert images from one format to another.

- **Image Resizer:** Access this feature to adjust image aspect ratios.

### Step 3: Using PicSqueeze Features

#### Image Compression

1. **Add Images:** Click the "Choose File" button to select the images you want to compress from your device's storage.

2. **Configure Settings:** Customize compression settings, such as quality or compression level.

3. **Compress:** Once you are satisfied with the settings, click the "Submit" button. The app will process the images and display the previous and current file sizes along with a download button for the compressed image.
   ![Image Compression Screenshot](https://i.postimg.cc/nLNZ1y7K/Screenshot-2023-09-09-224216.png)

#### Image Format Conversion

1. **Add Images:** Click the "Choose File" button to select the images you want to convert.

2. **Choose Output Format:** Select the desired output format from the available options.

3. **Convert:** Click the "Submit" button to initiate the conversion process. The app will provide a download link for the converted image in the chosen format.
   ![Image Format Conversion Screenshot](https://i.postimg.cc/7P2nHdNF/Screenshot-2023-09-09-225443.png)

#### Image Resizing

1. **Add Images:** Click the "Choose File" button to select the images you want to resize.

2. **Set Dimensions:** Specify the desired aspect ratio for the resized images.

3. **Resize:** Click the "Submit" button to adjust the images according to your specifications. The app will display a download link for the resized image.
   ![Image Resizing Screenshot](https://i.postimg.cc/wjZBjf2r/Screenshot-2023-09-09-225841.png)

#### Registration

1. **Fill Form:** Fill out the registration form with accurate information or register using your Twitter or Google account.

2. **Submit Form:** Click the "Submit" button to register on PicSqueeze and gain full access to all features.
   ![Registration Screenshot](https://i.postimg.cc/KcCjrFnz/Screenshot-2023-09-09-230837.png)
